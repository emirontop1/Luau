# LuaBank inspiration notes

The new site is positioned around two requests: a Lua obfuscator and a Lua bank similar in spirit to modern Lua whitelist platforms.

Research takeaways used in the design:

- Luarmor presents automatic integration, embedded whitelist checks, an ad key system, and a key-check API as core selling points.
- Luraph positions itself as a long-standing Lua obfuscator for securing scripts.
- Browser-based Lua obfuscator tools commonly advertise quick transforms such as local-variable renaming, string encoding, comment removal, whitespace flattening, and in-browser privacy.

LuaBank translates those ideas into a fresh static website: a dramatic hero, obfuscator demo, script-vault cards, key-flow explanation, and launch-focused pricing blocks.
